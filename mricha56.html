Beier-Neely Morphing

I used GIMP for Windows to make a looping .gif between the original F.bmp and F_scaled_2d.bmp.
See the Fs directory for the 10 constituent pictures.

The pictures were generated using the following bash script: 
#!/bin/bash

i=0

while [ $i -lt 10 ]; do
  ./Assignment1.exe --in mricha56_HTML/F.bmp --bnMorph mricha56_HTML/F_scaled_2d.bmp mricha56_HTML/f_multi_line_scale.txt 0.$i --out mricha56_HTML/Fs/$i.bmp
  echo 0.$i
  i=$[ $i + 1 ]
done
